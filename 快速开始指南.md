# 快速开始指南

## 5分钟快速上手指南

### 第一步：环境准备

1. **安装 Python 依赖**
   ```bash
   pip install PyQt6 opencv-python numpy ultralytics insightface pymysql
   ```

2. **准备 MySQL 数据库**
   ```sql
   CREATE DATABASE face_recognition;
   ```

3. **确认模型文件存在**
   - 确保 `yolov8x-face-lindevs.pt` 在项目目录中

### 第二步：配置数据库

编辑 `database.py` 第6行，修改数据库密码：

```python
def __init__(self, host="localhost", user="root", password="你的密码", database="face_recognition"):
```

### 第三步：启动程序

```bash
python UI.py
```

### 第四步：注册第一个人员

1. 点击 **"注册人脸"** 按钮
2. 点击 **"选择图片"** 选择照片
3. 输入姓名
4. 点击 **"注册人脸"**

### 第五步：测试识别

1. 点击 **"检测人脸"** 按钮
2. 选择包含已注册人员的图片
3. 查看识别结果和自动考勤记录

### 第六步：查看考勤记录

1. 点击 **"考勤记录"** 按钮
2. 查看所有考勤记录
3. 可以使用筛选功能查询特定人员或日期

## 功能按钮说明

| 按钮 | 功能 |
|------|------|
| 🆔 **注册人脸** | 注册新人员到系统 |
| 🔍 **检测人脸** | 识别图片中的人脸并自动记录考勤 |
| 🗑️ **删除管理** | 删除已注册的人员 |
| 📋 **考勤记录** | 查看和管理考勤记录 |

## 常见问题速查

**Q: 程序启动失败？**
- 检查模型文件是否存在
- 检查数据库连接配置
- 查看控制台错误信息

**Q: 识别不出来？**
- 确认该人员已注册
- 尝试使用更清晰的照片
- 检查照片中是否有人脸

**Q: 数据库连接失败？**
- 确认 MySQL 服务已启动
- 检查用户名密码是否正确
- 确认数据库已创建
